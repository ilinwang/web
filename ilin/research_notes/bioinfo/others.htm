There are four other comparison target algorithms in our paper, which are RTIP, PTG, HAPINFERX and SDPHapInfer.

RTIP

The Reduced TIP formulation for HIPP problem

RTIP is a program which exploits exact optimal algorithm (integer linear programming) to obtain the minimun number of haplotypes requried to infer a given genotype matrixdata.

Reference

Gusfield,D. (2003) Haplotype inference by pure parsimony, in Combinatorial Pattern Matching: 14th Annual Symposium,Lecture Notes in Computer Science, No. 2676, Springer Berlin / Heidelberg, 144-155.

Return

PTG

The Parsimonious Tree-Grow Method for HIPP problem

PTG is a heuristic algorithm that can find the minimum number of distinct haplotypes based on the criterion of keeping all genotypes resolved during tree-grow process.

Reference

Li,Z., Zhou,W., Zhang,X. and Chen,L. (2005) A parsimonious tree-grow method for haplotype inference, Bioinformatics, 21, 3475-3481.

Software Availability

The software is available upon request from the authors, or from http://zhangroup.aporc.org/bioinfo/ptg/ (only executable file can be obtained).

Return

HAPINFERX

The Clark's Inference Rule for HIPP problem

HAPINFERX is a program to infer haplotype from genotype data using a heuristic algorithm called Clark's inference rule.

Reference

Clark,A.G. (1990) Inference of haplotypes from PCR-amplified samples of diploid populations, Molecular Biology and Evolution, 7, 111-122.

Software Availability

The source cofe of HAPINFERX is available at http://www.nslij-genetics.org/soft/hapinferx.f.

Programming Language

HAPINFERX is written using Fortran.

Return

SDPHapInfer

The Semi-Definite Programming HapInfer for HIPP problem

SDPHapInfer is a program to infer haplotype from genotype data using semi-definite relaxation approximation algorithm to sovle the Integer Quadratic Programming formulation corresponding to the input genotype data .

Reference

Huang,Y.T., Chao,K.M. and Chen,T. (2005) An approximation algorithm for haplotype inference by maximum parsimony, Journal of Computational Biology, 12, 1261-1274.

Software Availability

The source cofe of SDPHapInfer is available at http://www.csie.ntu.edu.tw/~kmchao/tools/HapInfer/index.htm.

Programming Language

SDPHapInfer is written using Matlab and you have also install an external MatLab library called SDPA before excute it.
