MGP

The method of Merged Genotype Pairs for HIPP problem

MGP is a program to infer haplotype from genotype data using a heuristic algorithm which exploits the compatible relations among genotypes to solve a reduced integer linear programming problem in a smaller solution space.

How to use MGP?

Note! Since the program MGP is linked with CPLEX 9.0 callable library, you have to setup up your PC environment before excute it.

The steps for using MGP are as below.

  1. Download the executable file for Windows (compiled by Visual C++, and linked with CPLEX 9.0 callable library).
  2. Usage: MGP inputfile, for example, MGP b2ar.txt

The input file format

Input file contains the genotype data generated in experiment.
The first line specifies the number of genotypes and SNPs, i.e. genotype-number SNP-number.
One genotype is represented by a consecutive sequence of SNP sites, and each SNP site can be 0 (for homozygous wild type), 1 (for homozygous mutant), or 2 (for heterozygote).
A sample input file, b2ar.txt

The output file format

There are three output files after excuting MGP; two of them are integer programming formulations for the corresponding input file, and one of them is the inferred configurations for the input genotypes.

The first integer programming formulation for IPMPG1(C) is called MGP_inputfileStep1.lp, for example, MGP_b2arStep1.lp and the second integer programming formulation for IPMPG2(C) is called MGP_inputfileStep2.lp, for example, MGP_b2arStep2.lp.

The inferred configurations for genotypes are output in file MGP_inputfile.sol.
The first line contains: The number of selected haplotypes is: haplotype-number
One haplotype is represented by a consecutive sequence of SNP sites, and each SNP site can be either 0 (for homozygous wild type) or 1 (for homozygous mutant). Then resolutions of the input genotypes are listed.
The last line specifies the elapsed time for inferring the configurations for genotypes by GHI.
A sample output file, MGP_b2ar.sol

Reference

Wang, I. L. and Yang, H. E. (2006) Haplotyping populations by pure parsimony based on compatible genotypes submitted to Bioinformatics.
