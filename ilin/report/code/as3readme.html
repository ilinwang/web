The usage of as2 is the same as as3.

Differenc between as2 / as3 :

as2 as3
  • Label-Correcting Algorithm
    (Bellman-Ford Algorithm)
  • inefficient data structure, may do many inefficient search loops
  • do not comment much, hard to read
  • The shortest path function is easier to be understood, more straightforward. However, it's a little inefficient.
  • Label-Setting Algorithm
    (Dijkstra Algorithm)
  • better data structure, more efficient
  • include detailed comments, easily readable
  • The shortest path function uses binary heap. It may be hard to be understood, especially for some recursive functions

How to compile:

  • alias /opt/gnu/bin/gcc -ansi -O -wall -pedantic -o to be gcco
  • gcco as3 as3.c

How to execute:

  • as3 > path.out
