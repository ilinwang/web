This is the readme file for my first Assignment code : as4.c

  • File required for compile: as4.c as4.h
  • File required for run: topo.dat mino.dat
  • Output Files:
    • path.dat : a file stores the path information for each unit of demand (in the index of unit of demand)
    • arcpath.dat : a file stores the path information for each arc(virtual) (in the index of virtual arcs)
    • eq.dat : a file stores path information for each node, in 3 categories--
      • only out
      • transport
      • only in
    • vnode.dat : a file to store the associated information for each virtual node
    • varc.dat : a file to store the associated information for each virtual arc
  • Note:

How to compile:

  • alias /opt/gnu/bin/gcc -ansi -O -wall -pedantic -o to be gcco
  • gcco as4 as4.c

How to execute:

  • as4 > path.out
