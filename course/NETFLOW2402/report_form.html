General rules

  1. Written in Chinese or English
  2. For the paper version:
    1. You may use whatever editor (such as MSWord or Latex)
    2. <15 pages, 1~1.5 line spacing, font size≦12pt
  3. For the html/pdf version:
    1. Converted from (or copied/pasted from) your paper version
    2. If you use MSWord to make the webpages, remove all Word tags before you submit
    3. DO NOT use frames, only use one (long) page, create a directory named as YY_XX which contains all your files, zip the directory as YY_XX.zip, where XX=2p for 2 papers preview, XX=mp for midterm paper, YY is the number of your student ID (start with r , not R).
    4. Upload the zip file to the class web before the deadline

2 papers preview report format

  1. Beginning with your name/email, titles of these 2 papers (paper 1, paper 2)
  2. Paper 1:
    1. Title, authors, original abstract, where/when it was from
    2. Briefly explain why you choose this paper, what you expect to learn
    3. Decide whether you will use it as your midterm report or not
  3. Repeat above step for paper 2
  4. For html report, you should add links to those papers so that people can click to different paper
  • See sample pages

Midterm papers report format

Both the midterm paper/html reports should have the following format: (in this order)

  1. Your name, email, date
  2. Title, authors, original abstract, where/when it was from
  3. Content: (section 1, section 2, section 3, section 4)
    1. why you choose this paper
    2. what it did, what you have learned from it,…etc
    3. add any your viewpoints, comments
    4. list any other reference that you have used
  • You may use subsections or subsubsections inside each section.
  • For html report, you should add links to those sections so that people can click to different sections.
  • See sample pages

Presentation file format

  1. You may use powerpoint or whatever (even hand writing) for the presentation slides
  2. Whatever form you use, you should upload a pdf version electronically (you may use scan or print to create pdf files. If you don’t know how to do it, ask others.
