5th coding assignment
Due 05/19 (Thu) 08:00

You are asked to write 1program which can

  • read a network file with format like test2.sp
  • read an OD file with format similar to od1.txt (the first line represents total # of requested OD paris as represented in the remaining lines)
  • ask the user to choose algorithms to apply (1) DLU1 (2) DLU2 (either (1) or (2))
  • if (1) is choosen, it performs algorithm DLU1, which first calculates i0,j0 and k0, then perform A_LU, Acyclic_LU, and Reverse_LU. Finally it prints out the shortest path length for each requested OD pair (no need to trace the path)
  • if (2) is choosen, it performs algorithm DLU2, which performs A_LU and Get_P for each requested OD pair. Finally it prints out the shortest path (need tracing the entire path) together with its length for each requested OD pair.

Screen Input: network filename (e.g. test2.sp), requested od filename (e.g. od1.txt)
Screen Output:
1.if DLU1 is chosen, you have to print out the shortest path length for each requested OD pair
2.if DLU2 is chosen, you have to print out the entire shortest path (i.e. tracing the entire path) together with its length for each requested OD pair

Problems you may encounter:

1.if there exists no path from a requested Origin node to a requested Destination node, you should print out things like [there exists no such path]

2.how do you make sure your answers are correct? You may use the Floyd-Warshall algorithm that you have written last time to verify the results.

Special Notice:

It is your responsibility to check whether your algorithms work correctly or not. To do so, as suggested, you may use Floyd-Warshall algorithm to check the correctness of your answer. If you fail to do so and turn-in a buggy code which gives wrong answers, your grade will be discounted.
