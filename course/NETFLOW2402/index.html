開課班級： 工資管所碩博
教學時地： 授課：一6-8 [61206] ; Office hr: 四2~3 [61324] or by appointment
教學目標： Train students to learn network optimization and algorithms
課程簡介： We will first introduce foundations of graphs and algorithms, then go over each topic of network optimization.
教 科 書： Network Flows: Theory, Alg/orithms, and Applications by R. K. Ahuja, T. L. Magnanti, and J. B. Orlin,, 1993, Prentice Hall, NJ. ISBN: 0-13-617549-X
參考書籍：
  1. Introduction to Linear Optimization, by Bertsimas and Tsitsiklis, 1997, Athena Scientific, ISBN: 1-886529-19-1
  2. Combinatorial Optimization, by Papadimitriou and Steiglitz, 1998, Dover Pubns, ISBN: 0486402584
  3. Network Optimization: Continuous and Discrete Models, 1998, Athena Scientific, ISBN: 1-886529-02-7
  4. Linear Programming and Network Flows, 2nd Edition, by M.S. Bazaraa, J. J. Jarvis, and H.D. Sherali, John Wiley, ISBN: 0-471-63681-9

  1. Quiz (2 times: 2016/03/28, 2016/05/16, each time 10%) 20%
  2. Midterm Paper Presentation
    (decided by 2016/03/28 , submit by 2016/04/25 , present after 2016/04/25) 15%
  3. Final Exam (2016/06/20) 25%
  4. Homework (including programming assignments) 35%
  5. Class Participation 5% (extra points)
先修課程： Know how to program and some logic
對學生建議事項：
  1. Each student has to present some academic papers related with this course
  2. The computer program can be written by C, C++, or Java, but should run on unix environment (cygwin, Sun, Linux, or BSD)
預計進度：(16週)
  1. Introduction, Data Structure
  2. Graph Search, Decomposition
  3. Shortest Path: Label-Setting Algorithms
  4. Shortest Path: Label-Correcting Algorithms
  5. Max Flow Min Cut
  6. Min-cost Flow: Network Simplex Algorithms
  7. Assignments and Matchings
  8. Minimum Spanning Trees
  9. Review LP
  10. Lagrangian Relaxation
  11. Multicommodity Network Flows
