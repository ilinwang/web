6th coding assigment
Due 07/16 (Wed) 14:10

You are asked to write a program which can

  • read the following network file: mcf1.net
  • then, use network simplex algorithm (Chap 11) to solve the problem.

Screen Input: network filename (e.g. mcf1.net), s (source node)
Screen Output: optimal primal objective value, optimal dual objective value

example: suppose your code is named as netsimplex1, then when you execute netsimplex1, you should see:

Please input network filename: (then you should input mcf1.net)

then you should see the output like

optimal (primal, dual) objective: (then you should output 92, 92)

Note that thre may be more than 1 supply/demand node, and you will not know how many in advance, until you finish reading those lines begins with "n". Since this input is different from the shortest path one, you may have to spend time thinking how to dynamically read those supply/demand data.

What's difficult in this assignment may be

  1. you will have to modify your data structure used for solving the shortest path problem.
  2. you need to understand each step of network simpelx algorithm:
    1. how to get an initial Basic Feasible Solution (a spanning tree)
    2. how to identify a pivot-in nontree arc
    3. how to identify the negative cycle and compute flows along cycle
    4. how to update node potentials
